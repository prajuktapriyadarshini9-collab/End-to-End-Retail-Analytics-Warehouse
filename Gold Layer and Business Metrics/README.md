
# Gold Layer and Business Metrics

## Description
This folder contains the Gold Layer SQL models developed for the Retail Analytics Warehouse. The models provide analytics-ready datasets for dashboards and business reporting.

## My Contribution
- Developed Gold Layer SQL models.
- Created Fact and Dimension tables.
- Implemented business metrics:
  - Revenue
  - Average Order Value (AOV)
  - Customer Acquisition Cost (CAC)
  - Customer Lifetime Value (CLV)
  - Repeat Purchase Rate
  - Return on Ad Spend (ROAS)
- Validated SQL outputs in Snowflake.
- Prepared data for dashboard reporting.
